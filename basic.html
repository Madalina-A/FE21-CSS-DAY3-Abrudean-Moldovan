WarmUp Questions

You would like to use transitions and transformations in CSS3. What different naming conventions exists?

  • ease - specifies a transition effect with a slow start, then fast, then end slowly (this is the default)
  • ease-in - specifies a transition effect with a slow start
  • ease-out - specifies a transition effect with a slow end
  • ease-in-out - specifies a transition effect with a slow start and end
  • cubic-bezier(n,n,n,n) - lets you define your own values in a cubic-bezier function

    • Why is it useful to use services like caniuse.com?

    Caniuse.com is a website that allows us to search for whatever CSS method or attribute we want and make sure that the browsers support it. For example when looking for the term transform, we will immediately get recommended both the 2D and 3D transforms. Both are usable for most browsers, so we can feel secure in using this technology for our project.

    What are two kinds of elements in the flexbox model? What attributes do they have?

    Basically we are speaking about two different types of items in flexbox:
    The flex container (parent)
    The flex items (children)
    Flexbox consists of flex containers and flex items and defines how flex items are laid out inside a flex container.
    Inside a flex container there can be one or more flex items.
    Once an element is inside a flexbox container, it will follow the flexbox layout rules (instead of standard block & inline rules).
